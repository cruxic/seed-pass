--- jpm installation ---
sudo apt-get install nodejs-legacy npm
sudo npm install -g jpm

--- Firefox Setup ---

1) Install this addon
 https://addons.mozilla.org/en-US/firefox/addon/autoinstaller/

2) In about:config set
 xpinstall.signatures.required = false

3) In about:config add a new string value:
  extensions.sdk.console.logLevel=all

4) Create the initial xpi and then drag it to firefox
  jpm xpi

5) From here onward you can do
  jpm post --post-url http://localhost:8888/

Creating or importing a Seed should only be done on a secure computer. Please be sure that:
  • This computer has not been used by people you don't know or trust.
  • This computer is free of Viruses, Malware and Spyware.
  • This computer has all the latest security patches applied.
Before proceeding we recommend:
  1. Temporarily disable ALL of your browser plugins except for Seed-Pass.
  2. Disconnect from the Internet.
